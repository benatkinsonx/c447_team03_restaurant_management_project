# c447_team03_restaurant_management_project

## Project brief

This web application helps restaurant owners manage their business. It would handle their day to day online services such as:

- Showcasing their menu
- Handling takeaway orders
- Handling reservations
- Managing tables
- Authentication
- Reward scheme
- Merchandise
- Handle staff

## User stories

1. As a customer, I want to book a table for 3 people on Saturday, so that I can reserve a place to eat.
2. As a customer I want to view the menu
3. As a customer I want to receive rewards
4. As a customer, I want to create an account
5. As a customer I want to order food for takeaway
6. As an owner I want to be able to change the menu
7. As an owner I want to change staff roles
8. As a customer, I want to buy merchandise
9. As a customer, I want to cancel my reservation
10. As a customer, I want to amend my reservation
11. As a customer, I want to pay for my takeaway order
